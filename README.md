# Python Dynamic Island

A macOS Dynamic Island-style floating widget for Windows with transparent frameless window.

## Features
- ✨ Full transparency support (PySide6)
- 🎯 Frameless, always-on-top window
- 🎨 Hover animations
- 💻 Python backend with HTML/CSS frontend
- 📦 Open source (GPL v3)

## Installation
```bash
pip install PySide6 PySide6-Addons
python island_pyside.py
```
